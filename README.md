## Задача

Реализовать консольное приложение для игры [wordle](https://ru.wikipedia.org/wiki/Wordle). 

Приложение должно поддерживать следующие два позиционных аргумента командной строки:

**PATH** - путь к файлу словаря

**N**   -  длина загаданного слова

Аргументы идут последовательно в строгом порядке: `./app PATH N`

## Правила игры

Из всех слов, перечисленных в словаре, случайным образом выбирается слово длины N, после чего пользователю даётся 6 попыток угадать это слово. Каждую попытку нужно считывать слово из стандартного потока ввода, затем для каждой буквы введённого слова выводить её состояние в следующем формате:

| Статус | Описание                                                      |
|--------|---------------------------------------------------------------|
|  `-`   | Буквы нет в загаданном слове                                  |
|  `=`   | Буква есть в загаданном слове, но находится на другой позиции |
|  `+`   | Буква есть в загаданном слове и находится на той же позиции   |

В случае, если пользователь угадал загаданное слово, в консоль с новой строки выводится надпись `win!`, и игра заканчивается. В случае, если пользователь за 6 попыток не угадал, в консоль выводится фраза `The word was: ***`, где *** — это загаданное слово.

## Пример

Допустим, при запуске приложения командой `./app dictionary.txt 5` случайным образом было загадано слово `hello`. Игровая сессия может выглядеть следующим образом:
```
1. Игрок вводит слово: hatch
Ответ программы: +---=
(первая буква совпала по позиции)

2. Игрок вводит слово: coach
Ответ программы: -=--=
(вторая и последняя буквы есть в слове, но стоят в другой позиции)

3. Игрок вводит слово: kyoto
Ответ программы: --=-+
(последняя буква есть в слове и стоит в нужной позиции)

4. Игрок вводит слово: lllll
Ответ программы: ==++=
(буквы l есть в слове и стоят в нужной позиции)

5. Игрок вводит слово: spell
Ответ программы: --=+=
(третья и пятая буквы присутствуют, четвёртая — совпала по позиции)

6. Игрок вводит слово: hello
Ответ программы: +++++
win!
(Все буквы угаданы, игра выиграна)
```

## NB


### Пользовательский ввод
* Можно считать, что программу всегда запускают с корректными значениями агрументов
* Длина слов ограничена 100 символами
* Ввод не должен зависить от регистра. Некорректный ввод должен игнорироватья
* В случае, если слова заданной длины нет в словаре, в консоль выводится сообщение об ошибке и программа завершается с кодом 1

### Чтение из файла
* Делать более одного прохода по файлу не рекомендуется
* Длина слов ограничена 100 символами
* Символы только буквы
* Нужно игнорировать регистр слов
* Повторное чтение файла запрещено

### Полезные функции
* Для генерации случайных чисел можно воспользоваться функцией [std::rand](https://en.cppreference.com/w/cpp/numeric/random/rand.html)
